# 🛡 Cyber Sentinel

AI-Powered Cyber Threat Detection Platform built using Spring Boot, MySQL, HTML, CSS, JavaScript, and Bootstrap.

## 🚀 Features

- 🔐 Secure Login & Signup (BCrypt Password Encryption)
- 👤 Role-Based Access (Admin/User)
- 🛡 Fake Job Detection
- 🌐 URL Threat Detection
- 📜 Scan History
- 📊 Dashboard Analytics
- 📈 Charts & Statistics
- 📄 PDF Report Export
- 📊 Excel Report Export
- 👨‍💼 Admin Panel
- 🔍 Search Scan History
- 🗑 Delete Scan Records (Admin)
- 🚪 Logout

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/vradhichaudhary-web/CyberSentinel.git
```

2. Open in VS Code / IntelliJ

3. Create MySQL Database

```
cyberthreatdb
```

4. Configure `application.properties`

5. Run

```bash
.\mvnw spring-boot:run
```
